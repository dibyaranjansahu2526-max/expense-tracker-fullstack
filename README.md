# 💰 Full Stack Expense Tracker

A modern Full Stack Expense Tracker application built using Spring Boot and React. This application helps users manage their daily expenses efficiently with a clean and responsive user interface.

---

## 🚀 Features

- Add new expenses
- Update existing expenses
- Delete expenses
- View all expenses
- Search expenses
- Export expenses to Excel
- Responsive UI
- REST API Integration
- MySQL Database

---

## ⚙️ Installation

### Backend

```bash
cd ExpenseTracker
```

Run the Spring Boot application.

---

### Frontend

```bash
cd expense-frontend
npm install
npm run dev
```
